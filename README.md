## 🌟 Overview

An interactive fractal mandala generator featuring **4,000 points** with **7 distinct ring colors** and **ensemble learning** for anomaly detection. This visualization combines:

- ✨ Golden angle phyllotaxis (137.5°)
- 🌸 Polar rose pattern (k=8)
- 🕉 Delhi mandala (12-fold symmetry)
- 🎯 Power-law radial distribution ρ(r) ∝ 1/r^1.6
- 🧠 Ensemble learning (3 models: EWMA, Z-Score, IQR)
- 📊 Local fractal dimension analysis

## 🚀 Features

- **Interactive Controls**: Adjust point count, shift point, angle shift, and rose petals
- **7 Distinct Ring Colors**: Each concentric ring has a unique color
- **Real-time Visualization**: Mandala with ring-colored points
- **Statistical Analysis**: Distribution, fractal dimension, ensemble detection
- **Responsive Design**: Works on desktop and mobile devices
